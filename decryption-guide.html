Independent Decryption Guide

Your data is encrypted using standard, open algorithms — AES-256-GCM with PBKDF2-SHA256 key derivation. This means you can always decrypt your messages without this app, using tools available on any computer.

You will need: your original password and the Base64 encrypted string produced by the app.

Encrypted Data Format

When decoded from Base64, the blob has this exact layout:

BytesLengthContents
0 – 1516 bytesRandom salt (for PBKDF2 key derivation)
16 – 2712 bytesRandom IV / nonce (for AES-GCM)
28 – end-17variableCiphertext
last 1616 bytesGCM authentication tag

Key derivation: PBKDF2-SHA256, 100,000 iterations, 32-byte output.

Windows — PowerShell 7+

PowerShell 7 ships with .NET which includes built-in AES-256-GCM and PBKDF2-SHA256 — no extra software needed.

Check your version: Run $PSVersionTable.PSVersion in PowerShell. You need 7.0 or later. If you only have Windows PowerShell 5.x, download PowerShell 7 first.

Step 1 — Save your encrypted string

Paste the Base64 string from the app into a plain text file and save it as secret.txt in your home folder (e.g. C:\Users\YourName\secret.txt).

Step 2 — Run the decryption script

Open PowerShell 7, replace YOUR_PASSWORD_HERE with your real password, then paste and run the entire block:

# ── Replace with your actual password ──
$password = "YOUR_PASSWORD_HERE"

# ── Read and decode the Base64 blob ──
$bytes = [Convert]::FromBase64String((Get-Content "$HOME\secret.txt" -Raw).Trim())

# ── Split into components ──
[byte[]]$salt       = $bytes[0..15]
[byte[]]$iv         = $bytes[16..27]
[byte[]]$ciphertext = $bytes[28..($bytes.Length - 17)]
[byte[]]$tag        = $bytes[($bytes.Length - 16)..($bytes.Length - 1)]

# ── Derive the AES key with PBKDF2-SHA256 (100,000 iterations) ──
$pbkdf2 = [System.Security.Cryptography.Rfc2898DeriveBytes]::new(
    [System.Text.Encoding]::UTF8.GetBytes($password),
    $salt, 100000,
    [System.Security.Cryptography.HashAlgorithmName]::SHA256)
[byte[]]$key = $pbkdf2.GetBytes(32)
$pbkdf2.Dispose()

# ── Decrypt with AES-256-GCM (also verifies the auth tag) ──
$aesGcm    = [System.Security.Cryptography.AesGcm]::new($key)
$plaintext = New-Object byte[] $ciphertext.Length
$aesGcm.Decrypt($iv, $ciphertext, $tag, $plaintext)
$aesGcm.Dispose()

# ── Print the result ──
[System.Text.Encoding]::UTF8.GetString($plaintext)
Wrong password? AES-GCM will throw an AuthenticationTagMismatchException — this means the password is incorrect or the data was modified. No output is shown when authentication fails.

Mac / Linux — Python 3

Python 3 is pre-installed on macOS and most Linux distributions. You also need the cryptography package, which handles PBKDF2 + AES-GCM with full authentication.

Step 1 — Install the cryptography package (once)

pip3 install cryptography

Step 2 — Save your encrypted string

Paste the Base64 string from the app into a file:

pbpaste > ~/secret.txt      # macOS — pastes from clipboard
# or just: nano ~/secret.txt  and paste manually

Step 3 — Run the decryption script

Save the following as decrypt.py, replace YOUR_PASSWORD_HERE, then run it:

import base64
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.ciphers.aead import AESGCM
import os

# ── Replace with your actual password ──
password = b"YOUR_PASSWORD_HERE"

# ── Read and decode the Base64 blob ──
with open(os.path.expanduser("~/secret.txt")) as f:
    data = base64.b64decode(f.read().strip())

# ── Split into components ──
salt       = data[:16]
iv         = data[16:28]
ciphertext = data[28:]   # includes the 16-byte GCM tag at the end

# ── Derive the AES key with PBKDF2-SHA256 (100,000 iterations) ──
kdf = PBKDF2HMAC(
    algorithm=hashes.SHA256(),
    length=32,
    salt=salt,
    iterations=100000,
)
key = kdf.derive(password)

# ── Decrypt with AES-256-GCM (also verifies the auth tag) ──
plaintext = AESGCM(key).decrypt(iv, ciphertext, None)
print(plaintext.decode("utf-8"))

Run it from Terminal:

python3 decrypt.py
Wrong password? The cryptography library raises an InvalidTag exception — this means the password is incorrect or the data was modified. No output is shown when authentication fails.

One-liner alternative

If you prefer, you can run it inline without saving a file (replace the password first):

python3 - <<'EOF'
import base64, os
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.ciphers.aead import AESGCM

password = b"YOUR_PASSWORD_HERE"
data = base64.b64decode(open(os.path.expanduser("~/secret.txt")).read().strip())
salt, iv, ct = data[:16], data[16:28], data[28:]
key = PBKDF2HMAC(hashes.SHA256(), 32, salt, 100000).derive(password)
print(AESGCM(key).decrypt(iv, ct, None).decode())
EOF

Why This Works Without the App

This tool uses only open, standardised algorithms:

  • PBKDF2-SHA256 — defined in RFC 8018, available in every crypto library.
  • AES-256-GCM — a NIST standard, supported natively in .NET, Python, OpenSSL, Go, Rust, and more.

Any implementation that follows the format table above will decrypt your data correctly.
