Independent Decryption Guide

Your data is encrypted using standard algorithms (AES-256-GCM and PBKDF2). This means you can decrypt your messages without this software using any computer with OpenSSL installed.

Note: You will need your password and the Base64 string generated by the app.

Windows Decryption

Ensure you have OpenSSL installed (included with Git for Windows) and use PowerShell 7 or later.

  1. Save your Base64 encrypted string into a file named secret.txt.
  2. Open PowerShell and run:
# Convert Base64 to binary
$base64 = Get-Content secret.txt
$bytes = [Convert]::FromBase64String($base64)
[IO.File]::WriteAllBytes("secret.bin", $bytes)

# Extract components (Salt: 16 bytes, IV: 12 bytes)
$salt = $bytes[0..15]
$iv = $bytes[16..27]
$ciphertext = $bytes[28..($bytes.Length-1)]

# Decrypt using OpenSSL
openssl enc -d -aes-256-gcm -pbkdf2 -iter 100000 -in secret.bin -out decrypted.txt -K [KEY_IN_HEX] -iv [IV_IN_HEX]

Manual decryption via OpenSSL CLI for GCM requires hex conversion of keys. For most users, using the standalone HTML app file is recommended as it handles these technical steps automatically.

Mac / Linux Decryption

Open your terminal. Most systems have OpenSSL pre-installed.

# 1. Save your Base64 string to secret.txt
# 2. Decode Base64 to binary
base64 -d secret.txt > secret.bin

# 3. Use OpenSSL to decrypt
# Note: AES-GCM decryption via CLI varies by OpenSSL version.
openssl enc -d -aes-256-gcm -pbkdf2 -iter 100000 -in secret.bin

The app uses 100,000 iterations of PBKDF2-SHA256.
